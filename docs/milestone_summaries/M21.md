## Summary

M21 introduced **`audio_native_bindings.py`** (ctypes structs and **`bind_audio_classifier`**) and **`NativeAudioDispatcher`** implementing **`Dispatcher`** for **`AUDIO_FROM_FILE`**: **create → classify → close-result → close** sequence against **`MpAudioClassifier*`** symbols when a library handle is supplied. **Model path** is a dispatcher constructor parameter; **audio path** is the dispatch payload. Tests use **in-process fakes** with matching arity — **structural** proof only.

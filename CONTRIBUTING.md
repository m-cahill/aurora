# Contributing to AURORA

See [`DEVELOPMENT.md`](DEVELOPMENT.md) for prerequisites, editable install, tests, coverage, and what CI proves.

## Pull requests

- Open pull requests against the default branch; the required check **`ci` / `repo-safety`** must pass.
- Prefer small, reversible changes with clear acceptance criteria.
- Do not copy workspace-only governance (for example under a parent `docs/` tree outside this repository) into `aurora/` unless the material is intentionally public.
